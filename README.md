# 🤖 Agent-Based Autonomous Task Automator

A next-generation autonomous agent platform designed to streamline enterprise operations through intelligent task automation. The system accepts natural language commands, schedules background operations, manages task execution asynchronously, and provides real-time monitoring through a modern cyberpunk-inspired dashboard.

Built for the **Microsoft Agents League Hackathon**, this project demonstrates the practical application of AI-inspired agent workflows, multithreading, task orchestration, and robust exception handling using Core Java.

---

## 🚀 Features

* 🧠 Natural Language Command Processing
* ⚡ Asynchronous Task Execution
* 🔄 Multithreading with Thread Pools
* 📊 Real-Time Task Monitoring

## 🔄 Task Lifecycle

Every task progresses through the following stages:

```text
PENDING
   ↓
QUEUED
   ↓
RUNNING
   ↓
COMPLETED
```

In case of failures:

```text
FAILED
```

The agent automatically logs execution details and error information.
